\subsection{损耗与损坏}
在项目实施过程中，由于操作失误和元件质量问题，产生以下损耗：
\begin{itemize}
    \item 烧毁L298N模块1个（错误接线导致短路） - 损失8.90元
    \item 损坏STM32核心板1块（静电击穿） - 损失12.50元
    \item 杜邦线损耗约10根 - 损失1.50元
    \item \textbf{总损耗：} 22.90元（已计入总成本）
\end{itemize}
