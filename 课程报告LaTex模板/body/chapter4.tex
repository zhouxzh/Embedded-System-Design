\section{功能展示}

\subsection{功能描述}
本系统实现了基于红外遥控的智能风扇控制功能，主要包括基础控制功能和增强功能两大部分。

\subsubsection{基础控制功能}
\begin{itemize}
    \item \textbf{电源开关控制}：通过遥控器电源键实现风扇的一键开关机
    \item \textbf{三档风速调节}：支持低、中、高三档风速控制，通过PWM占空比调节电机转速
    \item \textbf{状态指示灯}：PG13（电源灯）、PG14（运行灯）实时反映系统状态
\end{itemize}

\subsubsection{增强功能}
\begin{itemize}
    \item \textbf{定时关闭功能}：支持1/2/4小时三档定时设置，定时结束自动关机
    \item \textbf{自然风模式}：模拟自然风变化，风速在30\%-70\%范围内随机波动
    \item \textbf{蜂鸣器反馈}：每次按键操作伴随蜂鸣提示音，增强交互体验
    \item \textbf{低功耗待机}：待机状态功耗<1mA，红外信号唤醒系统
\end{itemize}
