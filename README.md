# Embedded Systems Course Design

#### 介绍
本仓库为嵌入式系统课程设计提供参考资料和项目实例。包含教材、示例项目、实验指导、开发工具和讨论区。学生将获得嵌入式系统基础知识和实践经验。涵盖Arduino、STM32、Raspberry Pi、C/C++、Python等开发工具和语言。鼓励学生积极参与学习和讨论，培养解决问题能力。仅用于学术研究和教学。

#### 目录
1. 基于STM32的Boost电压变换器的设计
2. 基于STM32的反激变换器的设计
3. 基于STM32的SEPIC电压变换器的设计
4. 基于STM32的Buck电路的设计
5. 基于STM32的直流电机驱动
6. 基于STM32的收音机的设计
7. 基于STM32的万用表的设计
8. 基于STM32的可见光通讯系统的设计
9. 基于STM32的LED驱动电路的设计
